# Backend Project (E-commerce)

## 📌 Description

Spring Boot backend for an e-commerce application with REST APIs and MySQL integration.

## 🚀 Tech Stack

* Java
* Spring Boot
* MySQL
* JWT

## 📂 Features

* User Registration
* Login (basic/authentication if added)
* Database integration
* REST API development

## ⚙️ Setup Instructions

1. Clone the repository
2. Open in Eclipse
3. Configure MySQL in `application.properties`
4. Run the project

## 🔗 API Endpoints (sample)

* POST /register
* GET /users
